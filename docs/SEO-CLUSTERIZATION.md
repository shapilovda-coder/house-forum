# SEO-Кластеризация СтройСейлс

## 1. Структура URL (кластеры)

### Главная
```
/                              # Главная (Москва по умолчанию)
```

### Категории (ниши)
```
/category/prozrachnye-rolstavni/           # Прозрачные рольставни
/category/zashchitnye-rolstavni/           # Защитные рольставни  
/category/rolletnye-shkafy/                # Роллетные шкафы
/category/myagkie-okna/                    # Мягкие окна
/category/vorota/                          # Ворота
/category/bezramnoe-osteklenie/            # Безрамное остекление
/category/ofisnye-peregorodki/             # Офисные перегородки
```

### Регионы (гео)
```
/moskva/                                   # Москва
/sankt-peterburg/                          # Санкт-Петербург
/kazan/                                    # Казань
/krasnodar/                                # Краснодар
/ekaterinburg/                             # Екатеринбург
/novosibirsk/                              # Новосибирск
```

### Регион + Категория (пересечение)
```
/moskva/prozrachnye-rolstavni/             # Прозрачные рольставни в Москве
/sankt-peterburg/vorota/                   # Ворота в Питере
/kazan/myagkie-okna/                       # Мягкие окна в Казани
```

### Блог (информационные)
```
/blog/                                     # Список статей
/blog/kak-vybrat-prozrachnye-rolstavni/    # Статья
/blog/myagkie-okna-dlya-besedki/           # Статья
```

### Служебные
```
/kontakty/                                 # Контакты
/o-proekte/                                # О проекте
/politika-konfidentsialnosti/              # Политика
/sitemap.xml                               # Карта сайта
/robots.txt                                # Роботс
```

## 3. Title и Description шаблоны

### Главная
```
Title: СтройСейлс — поиск поставщиков рольставней, ворот и остекления
Description: Каталог проверенных поставщиков рольставней, ворот, мягких и безрамных окон в Москве и России. Сравнивайте цены и выбирайте лучших.
```

### Категория
```
Title: [Название категории] в Москве — 50+ поставщиков | СтройСейлс
Description: Каталог поставщиков [категория] в Москве. ⭐ Рейтинги, отзывы, цены. Сравните и выберите лучшего за 5 минут.
```

### Регион + Категория
```
Title: [Категория] в [Город] — поставщики, цены | СтройСейлс
Description: Ищете [категория] в [Город]? 20+ проверенных компаний. Рейтинги, отзывы, фото работ. Бесплатный подбор.
```

### Страница компании
```
Title: [Название компании] — [категория] | СтройСейлс
Description: ⭐ [Название компании] — поставщик [категория]. Контакты, цены, отзывы. Проверено СтройСейлс.
```

### Блог
```
Title: [Название статьи] | СтройСейлс
Description: [Краткое описание статьи]. Полезные советы от экспертов СтройСейлс.
```

## 4. Внутренняя перелинковка

### Хлебные крошки
```
Главная > Категория > Подкатегория > Страница
```

### Связанные страницы
- На странице компании: "Похожие компании"
- В категории: "Смотрите также (другие категории)"
- В статье: ссылки на категории и компании

### Footer-ссылки
- Популярные категории
- Популярные города
- Полезные статьи

## 6. Техническое SEO

### Обязательно
- [ ] HTTPS
- [ ] Адаптивность (mobile-first)
- [ ] Скорость загрузки < 3 сек
- [ ] ЧПУ (человеко-понятные URL)
- [ ] Canonical URLs
- [ ] Sitemap.xml
- [ ] Robots.txt

### Schema.org разметка
- Organization (организация)
- LocalBusiness (компании)
- Article (статьи)
- BreadcrumbList (хлебные крошки)
- FAQPage (вопрос-ответ)

### Мета-теги
- Title (уникальный, 50-70 символов)
- Description (уникальный, 150-160 символов)
- Open Graph (для соцсетей)
- Twitter Cards
