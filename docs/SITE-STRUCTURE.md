# СтройСейлс — Структура сайта

## Главная страница
**URL:** `/`

**Блоки:**
- Шапка с логотипом
- Hero-баннер с поиском
- Категории (7 штук)
- Список компаний
- Футер

## Категории

| URL | Название |
|-----|----------|
| `/prozrachnye-rolstavni/` | Прозрачные рольставни |
| `/zashchitnye-rolstavni/` | Защитные рольставни |
| `/vorota/` | Ворота |
| `/bezramnoe-osteklenie/` | Безрамное остекление |
| `/myagkie-okna/` | Мягкие окна |
| `/rolletnye-shkafy/` | Роллетные шкафы |
| `/ofisnye-peregorodki/` | Офисные перегородки |

## Гео-страницы

| URL | Описание |
|-----|----------|
| `/[category]/moskva/` | Категория в Москве |
| `/[category]/sankt-peterburg/` | Категория в СПб |
| `/[category]/[city]/` | Категория в любом городе |

## Вспомогательные страницы

| URL | Название |
|-----|----------|
| `/blog/` | Блог |
| `/blog/[article]/` | Статья |
| `/kontakty/` | Контакты |
| `/calculator/` | Калькулятор |
| `/404` | 404 ошибка |
