# Требования к карточке компании

## Обязательные поля

Каждая карточка компании **должна содержать**:

| Поле | Обязательность | Примечание |
|------|----------------|------------|
| **Название** | ✅ Да | Полное название компании |
| **Адрес** | ✅ Да | Физический адрес (если есть) |
| **Телефон** | ✅ Да | Контактный номер |
| **Сайт** | ✅ Да | Кнопка "На сайт" или "Рассчитать цену" |

## Приоритет отображения

### Первые две позиции (в ЛЮБОЙ категории)
1. **StekloRoll** (stekloroll-ru)
2. **Artalico** (artalico-ru)

Эти компании идут первыми **всегда**, независимо от:
- Выбранной категории
- Сортировки
- Фильтров

### Остальные компании
После StekloRoll и Artalico — остальные по:
- Кликам (популярность)
- Рейтингу
- Алфавиту

## Логика сортировки

```typescript
// ВСЕГДА первая
if (slug === 'stekloroll-ru') return -1

// ВСЕГДА вторая  
if (slug === 'artalico-ru') return 0

// Остальные по кликам/рейтингу
return clicks || rating
```
