## Поля карточки

| Поле | Формат | Пример |
|------|--------|--------|
| **Домен** | Только домен, без www, без https | `stekloroll.ru`, `artalico.ru` |
| **Телефон** | Любой: мобильный или городской | `+7 (903) 177-57-52` |
| **Адрес** | Полный адрес или пусто | `Москва, Киевское ш., БЦ Румянцево` |
| **Кнопка сайт** | Ссылка с UTM-меткой | `https://site.ru?utm_source=stroysales...` |
| **Кнопка телефон** | `tel:` ссылка | `tel:+79031775752` |

## UTM-метка для трекинга

```
utm_source=stroysales
utm_medium=referral
utm_campaign=catalog
```

Пример: `https://stekloroll.ru?utm_source=stroysales&utm_medium=referral&utm_campaign=catalog`

## Бейджи для StekloRoll и Artalico

- **StekloRoll**: "РЕКОМЕНДУЕМ" + "ПРОИЗВОДИТЕЛЬ"
- **Artalico**: "РЕКОМЕНДУЕМ" + "ПРЕМИУМ"

## Приоритет отображения

1. **StekloRoll** — всегда первая карточка
2. **Artalico** — всегда вторая карточка
3. Остальные — по кликам/рейтингу
