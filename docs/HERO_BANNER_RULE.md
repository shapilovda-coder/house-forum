# Требования к Hero Banner

## Правило
**Hero banner SVG — обязательный элемент главной страницы.**

Никогда не удалять, не заменять на простой цветной фон.

## Спецификация

### Файл
- **Название:** `hero-banner.svg`
- **Расположение:** `/public/hero-banner.svg`
- **Размер:** 1920×600 px (desktop), адаптивный

### Структура страницы
```
┌─────────────────────────────────────────┐
│              Header                     │
├─────────────────────────────────────────┤
│         Hero Banner SVG                 │
│    (illustration: house, shutters)      │
├─────────────────────────────────────────┤
│      H1 + H2 (text overlay area)        │
│   "Каталог поставщиков..."              │
├─────────────────────────────────────────┤
│           Search Bar                    │
├─────────────────────────────────────────┤
│         Company Cards                   │
├─────────────────────────────────────────┤
│           Footer                        │
└─────────────────────────────────────────┘
```

### Цвета баннера
| Элемент | HEX |
|---------|-----|
| Основной синий | #3B82F6 |
| Тёмно-синий | #1E40AF |
| Акцент оранжевый | #F97316 |
| Фон | #F3F4F6 → белый |

### Контент баннера
- Абстрактный дом/фасад
- Рольставни (прозрачные)
- Ворота автоматические
- Окна / остекление
- **Место для текста слева (60% ширины)**

### Запрещено
- ❌ Удалять баннер
- ❌ Заменять на сплошной цвет
- ❌ Менять порядок: баннер → текст → поиск
- ❌ Перемещать H1/H2 выше баннера

## Код в HomePage

```tsx
{/* Hero Banner SVG — НЕ УДАЛЯТЬ */}
<div className="relative w-full">
  <img src="/hero-banner.svg" alt="" className="w-full h-auto" />
</div>

{/* H1 + H2 Text — ПОСЛЕ баннера */}
<div className="bg-gradient-to-r from-blue-600 to-blue-700 py-8 px-4">
  <h1>Каталог проверенных поставщиков...</h1>
  <p>800+ компаний...</p>
</div>
```

## Проверка при деплое
- [ ] Баннер загружается (HTTP 200)
- [ ] H1/H2 после баннера
- [ ] Поиск под текстом
- [ ] Нет ошибок в консоли
