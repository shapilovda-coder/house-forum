# ⚠️ КРИТИЧЕСКОЕ ПРАВИЛО: Деплой

## ЗАПРЕЩЕНО

❌ Деплоить каждые 3 секунды
❌ Деплоить без проверки
❌ Деплоить напрямую в production
❌ Делать больше 5 деплоев/час

## РАЗРЕШЕНО

✅ Максимум 5 деплоев/час
✅ Только после проверки на localhost
✅ Только через staging → production
✅ С паузой минимум 10 минут между деплоями

## ПОСЛЕДСТВИЯ НАРУШЕНИЯ

- Блокировка Vercel
- Потеря дня работы
- Потеря доверия пользователя
