- **Лимиты:** 100 GB трафика/мес

### 2. Supabase (база данных)
- **Цена:** $0
- **Лимиты:** 500 MB, 2 GB трафика

- **Хватает:** Да

### 3. Бэкапы (ручные)
- **Цена:** $0
- **Как:** Раз в неделю экспорт JSON
- **Где:** Локально + Google Drive
