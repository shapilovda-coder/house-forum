# SEO Руководство

## 1. Стратегия

Многостраничный сайт с отдельными URL для каждой комбинации категория + город.

**Принцип:** Не фильтр на главной, а **отдельные страницы** для SEO.

## 2. Структура URL

```
/                                    # Главная (все категории, все города)
/prozrachnye-rolstavni/              # Категория (вся Россия)
/prozrachnye-rolstavni/moskva/       # Категория + Москва
/prozrachnye-rolstavni/sankt-peterburg/  # Категория + СПб
```

## 7. Генерация страниц

```typescript
export async function generateStaticParams() {
  const categories = ['prozrachnye-rolstavni', 'zashchitnye-rolstavni', ...]
  const cities = ['moskva', 'sankt-peterburg', 'novosibirsk', ...]
  
  const paths = []
  
  // Главная
  paths.push({ slug: [] })
  
  // Категории
  categories.forEach(cat => {
    paths.push({ slug: [cat] })
  })
  
  // Категория + город
  categories.forEach(cat => {
    cities.forEach(city => {
      paths.push({ slug: [cat, city] })
    })
  })
  
  return paths
}
```
